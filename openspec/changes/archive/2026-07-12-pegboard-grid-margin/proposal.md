## 为什么

App 预览与用户手里的**实体拼豆板对不齐**,两个具体问题:

1. **交互预览缺每 10 格的加粗分割线**——导出的引擎 `gridPng` 有(细网格 + 每 10 格粗线 + 行列号),
   但结果页的交互格子视图(`bead_grid_view`)只画均匀细网格,大图(如 100×100)数「第几行第几格」很吃力。
2. **实体拼豆板有「辅助网格 + 边框圈」而 App 不体现**——国产刻度拼豆板(如离草「104 长钉刻度板」)=
   印刷辅助网格(整数位 50 / 100)+ 辅助线外 1–2 圈边框钉位(→ 52 / 104)。用户照板拼时需要「屏幕行列号
   对上板上刻度」+「板的边框留白」,但当前 App 无边框选项、也不显示对齐板的 1-based 刻度。照 App 拼实体板
   的体验因此割裂。

这是 M10 把 App 推向「像成品」的一环。

## 变更内容

- **App 接管用户可见的图纸渲染**:交互预览(`bead_grid_view`)与「保存到相册」的图**都由 App 从
  `BeadPattern.cells` + palette 自渲染**(共用同一**几何/布局**、加防漂移测试),加:**每 10 格加粗分割线**
  + **1-based 内容刻度**(对齐实体板辅助网格,轴上每 10 标号,内容格 = 1..W / 1..H,**非方形亦然**;`W/H<10`
  该轴无刻度)。**两层几何**:最终画布 = 刻度 margin + 板面逻辑区。
- **可配边框圈 k(板对齐留白)**:生成前设默认值(**并入既有持久化需求**)+ **结果页可实时改**(`k` 状态提到
  `ResultPage` 级,两个保存入口共享);内容 `W×H` → 板面 `(W+2k)×(H+2k)`,边框为**空留白、无珠、不入统计、
  不进 `BeadPattern`、不编号**,**点边框区 no-op**。`k` **默认 0**、有硬上限(clamp 旧值)。
- **「保存到相册」改 CPU 光栅化(`image` 包)出图**(含分割/刻度/边框),**不用 `Picture.toImage`**(撞既有裁剪
  需求禁令 + iOS 模拟器软渲染 + GPU 纹理上限);保存图有**像素预算**(每格像素随尺寸自适应降,防超大图 OOM)。
  不再存引擎 `gridPng`。
- **引擎 / CLI 零改动**:`bead-core` 保持纯净,`gridPng` 降为开发辅助;统计取 `output.stats`、`BeadPattern`
  仍是唯一真相源(不破硬规则 3/5)。

## 非目标

- **不动 `bead-core` / `bead-cli` / golden / 确定性**——纯 `apps/mobile` 展示层;引擎 `render_grid` 的
  `gridPng` 保持现状(仍作开发辅助,不加边框/不改编号)。
- **不改 `BeadPattern` 数据 / 统计**——边框是空留白钉位、无珠子,绝不进 `BeadPattern`、不污染统计。
- **不建「板子产品库」**——各家板规格不同,只给通用「边框圈数 k」;不做「选具体板 → 自动套辅助尺寸/圈数」
  (用户判断大概率不需要)。
- **不做多板拼接 / 板号页码**——大图跨多板的 overlap / 索引页范式留待后续(本次单板对齐)。
- **不改保存相册的权限/首保提示/安全区**等既有行为——只换「保存什么图」。

## 功能 (Capabilities)

### 修改功能

- **mobile-app**:`结果页可保存到相册`——保存内容由引擎 `gridPng` 原文改为 **App 用 `image` 包 CPU 光栅化
  自渲染的图纸**(含每 10 格分割 / 1-based 刻度 / 可配边框圈);反转原「禁止壳内重绘」约束、禁 `Picture.toImage`。
- **mobile-app**:`结果页逐格可交互格子视图`——letterbox 宽高比由「保持 pattern 宽高比」改为「按渲染画布
  `canvasAspect`（含刻度 margin + 板面 `(W+2k)×(H+2k)`)」(margin 在任一轴≥10 或 k>0 时即 ≠ W:H;格子仍正方
  不拉伸);消除与新增需求的合并矛盾。
- **mobile-app**:`设置页配置跨启动持久化`——持久化字段清单纳入 `borderRings`(默认 0)。

### 新增功能

- **mobile-app**:`预览与保存图对齐实体拼豆板（每 10 格分割 + 1-based 刻度 + 可配边框圈）`
